# SupplyChain360 Analytics

## 📌 Project Overview

SupplyChain360 Analytics is an end-to-end data analytics project that analyzes supply chain operations for a global retail company.

This project uses Python, SQL, Statistics, Machine Learning, Excel, and Power BI to identify operational inefficiencies, improve delivery performance, optimize inventory, and support business decision-making through data-driven insights.

---

## 🎯 Business Problem

The company is facing several operational challenges:

- Increasing delivery delays
- Inventory shortages and overstock situations
- Rising logistics costs
- Low profit despite high sales
- Lack of centralized reporting

This project aims to analyze historical supply chain data and provide actionable business recommendations.
